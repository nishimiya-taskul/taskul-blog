# TASKUL Blog SEO運用ログ

Claudeが行った全ての分析・判断・行動・結果を記録する場所です。

## ファイル構成
- `YYYY-MM-DD.md` — 日次ログ（その日に行った作業の記録）
- `articles/` — 記事ごとの分析・改修履歴
- `monthly/` — 月次レポート（GSCデータ集計、リライト判断）

## 日次ログのフォーマット
```markdown
# YYYY-MM-DD SEO運用ログ

## 今日の作業サマリー
（1〜3行でその日の概要）

## 実行した作業

### [作業名]
- **意図**: なぜこの作業をしたか
- **分析**: 何を調べて何がわかったか
- **行動**: 具体的に何をしたか
- **結果**: どうなったか（公開直後は「計測待ち」でOK）
- **次のアクション**: この作業の次に何をすべきか

## 記事ステータス一覧
| 記事 | 公開日 | ターゲットKW | 現在順位 | 表示回数 | CTR | 次のアクション |
|------|--------|------------|---------|---------|-----|--------------|
```
